 # Tic Tac Toe Game

A simple and engaging Tic Tac Toe game built using **HTML, CSS, and JavaScript**.


##  Project Overview

This project implements the classic Tic Tac Toe game running directly in
the browser.\
It features a clean UI, full game logic, win detection, and a restart
option.

##  Features

-   Modern and responsive user interface\
-   Full game logic including:
    -   Win detection\
    -   Draw detection\
-   Prevents clicking the same cell twice\
-   Popup window for game result\
-   Restart button without refreshing the page

##  Project Structure

    Tic_Tac_Toe/
    │── index.html      # Main structure of the page
    │── style.css       # UI styling and layout
    │── script.js       # Game logic and functionality
    └── assets/         # (Optional) Additional images or files

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **JavaScript**

##  How the Game Works

-   Each click fills the selected cell with "X" or "O".\
-   Turns automatically alternate between players.\
-   After every move, the script checks all winning patterns.\
-   If a win or draw is detected, a popup is displayed.\
-   Clicking the **Restart** button resets the game.

##  How to Run

Just open the `index.html` file: 1. Download the project\
2. Open `index.html` in your browser\
3. Start playing!
